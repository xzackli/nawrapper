NaWrapper¶

NaWrapper is a thin python wrapper around NaMaster for CMB work, in particular for the ACT experiment.

It bundles all the necessary pieces associated with a map (i.e. beams, masks, filters) into a nawrapper.power.namap_hp (for HEALPix maps) or nawrapper.power.namap_car (for rectangular pixels). It also provides concise utility functions for turning these into dictionaries of power spectra.
